We have made a few changes in the MARK5 app. Scroll down to see what has been updated:

1. CUSTOMIZABLE SWIPE ACTIONS

You can now customize swipe actions. To customize swipe actions, go to settings and select "Swipe options" under emails.

2. SELECT ALL EMAILS ACCESSIBLE IN APP

You can now select all emails in the folder you are working in. To do so, long press on any email. A selection view will appear with a button in the top right corner. Tab it to select all so you can edit everything at once.

3. INSERT TEMPLATES WHERE THE CURSOR IS

When you are writing an email, place the cursor where you wish to insert a template in the text. Click the attach icon and insert a template and it will appear where you placed the cursor.

4. CALL ID

You now have the option to see the names of MARK5 contacts on your phone if they call you. To do so, swipe left on the folder where the contacts are placed. Select “Save offline” and enable Call ID in settings. Please notice that in order for Call ID to work you need to add the country code to the phone numbers.

5. VARIOUS FIXES AND IMPROVEMENTS

We hope that you are happy with this new version of MARK5. If you discover any bugs, please do not hesitate to contact us and we will look into it as soon as possible.
